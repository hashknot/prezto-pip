prezto-pip
==========

Provides [prezto][1] autocompletion support for [Pip][2].
Based on [oh-my-zsh][3] pip autocompletion [plugin][4].

Functions
---------

  - `_pip` supports autocompletion for pip
  - `pip-clear-cache` clears cached list of pip packages
