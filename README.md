Pip
===

Provides autocompletion support for [Pip][1].
Based on [oh-my-zsh][2] pip autocompletion [plugin][2].

Functions
---------

  - `_pip` supports autocompletion for pip
  - `pip-clear-cache` clears cached list of pip packages
